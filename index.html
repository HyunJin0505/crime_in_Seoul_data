Intro

Purpose of Project

수많은 사람들이 살아가고 있는 도시 서울에서 발생하는 5대 범죄 사건들에 대한 데이터를 수집하고 구 별 범죄 현황을 확인한다. 파이썬 데이터분석 라이브러리들을 활용하며 학습한다.
분석한 내용을 시각화하여 정보 파악이 용이하게 한다.

1. Data Analysis

데이터 전처리를 위해 사이킷런 preprocessing 패키지를 사용한다.
데이터 가공과 편짐을 위해 Pandas, Numpy 라이브러리를 활용한다

2. Data Visualization

파이썬 데이터 시각화 라이브러리 Seaborn과 지도 시각화 라이브러리 Folium을 활용
Seaborn을 통해 pairplot, heapmap 형태 시각화 구현
Folium을 사용하여 서울시 경찰서별 검거율과 구별 범죄 발생빈도 시각화 주소, 서울시 경찰서 위치 지도 구현

Data Analysis

Pandas, Numpy

데이터를 각 구별로 나누어 정리했다.
각 구별 데이터 - 5대 범죄 각 발생건수, 검거율, 인구수, CCTV개수, 총 범죄, 총 검거율
비교에 용이하도록 범죄와 검거율 총합은 0~1사이의 소숫로 정규화했다.

Notice
✓ 인구수의 경우 세부적인 구분 없이 명 수만으로 계산했다.

Pairplot

강도, 살인, 폭력의 관계

Seaborn 라이브러리를 통해 Pairplot 형태로 시각화 구현
Pairplot의 경우 'scatter','kde','hair','reg'네 종류의 그래프를 지원한다.
이 경우에는 reg 옵션을 선택했다.

  • 시각화 결과: 강도, 살인, 폭력은 양의 관계를 갖고 있다.

Word Cloud

word cloud 라이브러리를 사용하여 구현
사용한 데이터에서 빈도가 숫자(합계, 비율)로 표현되어 있어서 워드클라우드 구현에 활용하지 못했다.
5대 범죄 키워드로 워드 클라우드를 생성했으나, 이후 숫자형 데이터를 이용한 구현 방법에 대해 연구할 계획이다.

Heapmap

구별 경찰서의 검거율

Seaborn라이브러리를 통해 데이터를 heatap 형식으로 시각화했다.
색감의 농도를 통해 크기, 빈도 등의 정보를 파악할 수 있다.

이 경우에 절도의 검거율이 다른 5대 범죄의 검거율보다 현저히 떨어진다는
사실을 파악할 수 있다.
검거율이 가장 높은 3구는 도봉구, 금천구, 강서구 이며
가장 낮은 3구는 중구, 동작구, 구로구라는 사실을 확인할 수 있다.

Heapmap

5대 범죄 발생량

Seaborn라이브러리를 통해 데이터를 heatap 형식으로 시각화했다.
색감의 농도를 통해 크기, 빈도 등의 정보를 파악할 수 있다.

범죄 건수를 0~1사이의 소수로 정규화하여 비교가 용이하도록 만들었다.
범죄 발생 건수의 비율이 가장 높은 3구는 강남구, 영등포구, 송파구 이며
가장 낮은 3구는 도봉구, 서대문구, 금천구라는 사실을 확인할 수 있다.
